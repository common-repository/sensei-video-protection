== Description ==

Woothemes Sensei  + Fv Flowplayer + Sensei video protection

If you use the Woothemes Sensei plugin you will always worry about someone get my video download illegally. Users can right-click and run the source view to get the URL of the video in the lesson page. Sensei video protection plugin make it difficult a general user download the video. With this lesson page is running in a browser without toolbars. Of course user can not run the browser¡¯s menu, and browser extensions because no Toolbar. It also block mouse-right-click and keyboard almost completely. But this is not enough. User can try different ways to view the source. But if you use the fv flowplayer plugin you do not worry about it. User can not find your video url in source because video url is encrypted by Sensei Video Protection.

Features

    Secure Lesson Window ? No Toolbar, No Browser Menu, No View-Source, No Extension
    Prevent direct lesson page access (login user)
    Prevent lesson page open if disable javascript
    Prevent Right Click & Keyboard & Drag (Pro version only)
    Detect Chrome video download extension (Pro version only)
    Video URL Encryption ? View-Source Encryption - Only fv Flowplayer embed (Pro version only)

To purchase the Pro version, click on the following link : http://senseivp.com


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/sensei-video-protection` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->sensei security screen to configure the plugin
